About the data

I captured the raw data which goes into our WoW Audit sheet every hour throughout the season and dumped it into a MongoDB timeseries collection. You can see this data in the first many rows of the raw_data tab.

I’ve done some minor data cleanup, and computed “game weeks” and “game days” for many of these plots. Game days are assumed to start at 7am, except for Tuesdays. The game week starts based on a best-effort recognition of the weekly reset.
